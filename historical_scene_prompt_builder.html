What this tool actually does
This page does not retrieve any historical data itself — it only builds a research instruction. Whether real facts get used depends on which mode you picked above. "Use real historical data" mode asks Claude to search and cite sources. "Conceptual reconstruction" mode explicitly tells Claude not to search and to label everything as speculative. Either way, the weather field is always treated as an artistic mood choice, never a verified historical record, unless Claude finds an actual dated weather record and says so.
